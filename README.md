## üì¶ Installation

1. **Clone the repository**

   ```bash
   git clone <repository-url>
   cd dot-web-hosting
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Run the development server**

   ```bash
   npm run dev
   # or
   npm run dev -- --port=3001 # if port 3000 is in use
   ```

4. **Build for production**

   ```bash
   npm run build
   npm start
   ```
